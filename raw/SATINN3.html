Tubule Outputs

The data table to the right is the primary output of this workflow. All other figures can be recreated using the input images, native segmentation (if applicable), and the data table. The preview only shows a small subset of the full table to reduce load times.

  • Save will save the full data table, as *.csv or *.txt.

The bottom figures are samples that can be recreated using the inputs and data table. You can use this to verify the accuracy of segmentation and classification. Only segmented objects can be classified. Tubules were initially filtered with a generic size and circularity exclusion filter. If tubules are missing or inaccurate, it is usually due to low quality microscopy images, incorrect segmentation parameters, or unsuitable filtering.

  • Segmentation image. This is the output of native tubule segmentation, post-filtering. Cells must be contained in one of these tubules to be considered 'in-tubule' in the 'Cells' output tab.
  • Classification image. Each tubule class is color coded based on the legend (far right). Each color represents the tubule class of the highest confidence.
  • Confidence distribution. This is a normalized barplot of the confidence distributions (x-axis) for each tubule class (y-axis).
