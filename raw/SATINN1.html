Input Instructions

Select the image channels you wish to analyze by clicking on the following buttons:

Required:

  • Hoechst: Nuclear signal required for cell classification.
  • ACTA2: Actin-alpha channel required for tubule segmentation. This will be performed natively, using Otsu's method.
  • ACRV1: Acrosomal vesicle channel required to assist tubule classification, and may be used for cell classification.

Optional:

  • Hoechst (Segmentation): User-specified cell segmentation image. If no image is specified here, cell segmetation will be performed natively.
  • ACTA2 (Segmentation): User-specified tubule segmentation image. If no image is specified here, tubule segmentation will be performed using native parameters and size and circularity filtering.

The dots next to each input channel will turn green when they have been imported successfully.

Next, choose the neural network you wish to use for cell classification. Pre-trained networks include the network used in the Yang et al., 2022 manuscript, or if you have created your own using Matlab's Deep Learning Toolbox, you may specify it by selecting Custom. If you choose this option, the file must be a .mat file that includes cell classification as the `net` variable. All other variables will be ignored.

Once you have selected the images and neural networks, click Process. It is recommended that you start with a small test image set, as large ones can take hours of processing time.
